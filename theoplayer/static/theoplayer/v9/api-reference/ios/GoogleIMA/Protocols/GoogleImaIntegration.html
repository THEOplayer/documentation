    The ads rendering settings which will be used when requesting an ad.

    Remark

    IMA will use this property as the primary rendering settings. The rendering settings can also be provided with each ad defined by NativeGoogleImaAdDescription, GoogleImaIntegration.requestAds or GoogleImaIntegration.schedule as fallback in case this property is nil.
