# SDKs

Dolby OptiView Player provides a set of feature-rich video player software development kits (SDKs). Each SDK targets a set of platforms. For example, a developer integrates the iOS SDK in their iOS app.

Dolby OptiView Player offers the following SDKs. (Clicking the link takes you to the respective getting started guide.)

* HTML5/Tizen/webOS SDK

  * [Web](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/web/getting-started.md) targets (desktop and mobile) browsers, HTML5-based smart TVs, progressive web apps, and other HTML5-based platforms.
  * [webOS](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/webos/getting-started.md) targets LG webOS apps.
  * [Tizen](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/tizen/getting-started.md) targets Samsung Tizen apps.
  * [Chromecast](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/chromecast/getting-started.md) targets (custom) Chromecast apps.

* Android/Android TV/Fire TV SDK

  * [Android](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/android/getting-started.md) targets Android, Android TV and Fire TV apps. It may also be used for other Android-based platforms.

* iOS/tvOS SDK

  * [iOS & tvOS](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/ios/getting-started.md) targets iOS, iPadOS and tvOS apps.

* Roku SDK

  * [Roku](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/roku/getting-started.md) targets Roku apps.

* [React Native](/documentation/pr-preview/pr-690/theoplayer/getting-started/frameworks/react-native/getting-started.md) SDK

Want to update an existing SDK instead? Follow [this guide](/documentation/pr-preview/pr-690/theoplayer/getting-started/sdks/how-to-update-a-sdk.md).

Looking for the Legacy Android and iOS/tvOS (4.12.x) SDKs?

The legacy Android and iOS/tvOS SDKs (version 4.12.x) are no longer supported. However, you can still access their documentation through the following links:

* Android/Android TV/Fire TV Legacy (4.12.x) SDK

  * [Android](/documentation/pr-preview/pr-690/theoplayer/v4/getting-started/sdks/android/getting-started/) targets Android apps, and may be used for other Android-based platforms.
  * [Android TV](/documentation/pr-preview/pr-690/theoplayer/v4/getting-started/sdks/android-tv/getting-started/) targets Android TV apps.
  * [Fire TV](/documentation/pr-preview/pr-690/theoplayer/v4/getting-started/sdks/fire-tv/getting-started/) targets Fire TV apps.

* iOS/tvOS Legacy (4.12.x) SDK

  * [iOS](/documentation/pr-preview/pr-690/theoplayer/v4/getting-started/sdks/ios/getting-started/) targets iOS and iPadOS apps.
  * [tvOS](/documentation/pr-preview/pr-690/theoplayer/v4/getting-started/sdks/tvos/getting-started/) targets tvOS (i.e. Apple TV) apps.
