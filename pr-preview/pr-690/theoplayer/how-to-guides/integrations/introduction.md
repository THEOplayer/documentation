# Introduction

In this section, you can figure out how THEOplayer integrates with various partners.

<!-- -->

## [📄️IRIS.TV](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/iris-tv.md)

[Iris.TV is a recommendation engine. When hooking THEOplayer to the IrisTV SDK, a UI will be overlaid over the default player UI which can be used to navigate through recommendations.](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/iris-tv.md)

## [📄️Wowza streaming engine with secure token version 2](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/wowza-streaming-engine-with-secure-token-version-2.md)

[This guide explains how to set up THEOplayer with Wowza Streaming Engine™, using their SecureToken version 2 module, that provides playback protection via a security token that is exchanged between the server and clients. The SecureToken is a challenge/response system that helps to protect content against spoofing threats. Each connection is protected by a random key and a password (shared secret).](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/wowza-streaming-engine-with-secure-token-version-2.md)

## [📄️Deliver Low-Latency HLS live streams using Wowza Streaming Engine and THEOplayer](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/deliver-low-latency-hls-live-streams-with-wowza-streaming-engine.md)

[Wowza Streaming Engine™ media server software version 4.7.8 and later together with THEOplayer version 2.65 and later can deliver Low-Latency HLS (LL-HLS) live streams.](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/integrations/deliver-low-latency-hls-live-streams-with-wowza-streaming-engine.md)
