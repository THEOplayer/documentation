## [📄️Clipping](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/miscellaneous/clipping.md)

[THEOplayer supports playing only a selected part of a video-on-demand stream, rather than playing it from start to finish. This can be used for example to watch a single game-changing event from a sports match, highlight an important part of a long video presentation, or skip straight to the best part of a song's video clip.](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/miscellaneous/clipping.md)
