# Yospace

Yospace is a Server-Side Ad Insertion solution. A demo for the Web SDK can be found [here](https://demo.theoplayer.com/ssai).

##### Web SDK[​](#web-sdk "Direct link to Web SDK")

You need to set up [THEOplayer Yospace Connector for Web](https://github.com/THEOplayer/web-connectors/tree/main/yospace). Please refer to [this](/documentation/pr-preview/pr-690/theoplayer/connectors/web/yospace.md) section to learn all about how to integrate and use the Yospace connector on Web SDK.

##### Android (TV) SDK[​](#android-tv-sdk "Direct link to Android (TV) SDK")

You need to set up [THEOplayer Yospace Connector for Android](https://github.com/THEOplayer/android-connector/tree/master/connectors/yospace). Please refer to [this](/documentation/pr-preview/pr-690/theoplayer/connectors/android/yospace.md) section to learn all about how to integrate and use the Yospace connector on Android (TV) SDK.

##### iOS (/tvOS) SDK[​](#ios-tvos-sdk "Direct link to iOS (/tvOS) SDK")

You need to set up [THEOplayer Yospace connector for iOS](https://github.com/THEOplayer/iOS-Connector/tree/main/Code/Yospace). Please refer to [this](/documentation/pr-preview/pr-690/theoplayer/connectors/ios/yospace.md) section to learn all about how to integrate and use the Yospace connector on iOS (/tvOS) SDK.

## Conclusion[​](#conclusion "Direct link to Conclusion")

THEOplayer partnered with Yospace to fully integrate their Server-Side Ad Insertion solution, allowing you to play their streams in a breeze.

This partnership means that we are compatible with each other: THEOplayer has a long history with and is optimized to play Yospace streams. We regularly validate our connectors and are the first to know about changes to the Yospace infrastructure.
