# Introduction

The articles in this section describe how to implement common use-cases related to advertisements. These use-cases can range from customizing the UX during advertisements, to detecting ad blockers.

We recommend starting with [How to set up VAST and VMAP ads](/documentation/pr-preview/pr-690/theoplayer/how-to-guides/ads/how-to-set-up-vast-and-vmap.md), as this article explains how to set up client-side advertisements.

Consider reading our [Advertising User Guide](/documentation/pr-preview/pr-690/theoplayer/knowledge-base/advertisement/user-guide.md) to learn more about advertising and THEOplayer's capabilities.
