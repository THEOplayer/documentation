# List Stream Summary

List a summary of all streams created within last hour with specific sorting and pagination. If the transcoderId is provided, all streams associated with the current or last active instance will be returned, regardless of the stream creation time. If response array is empty, you have reached the end of the list ordering. This is a lite version of [List Streams](/documentation/pr-preview/pr-690/millicast/api/monitoring-list-recent-streams.md) which does not include feed details.
