# List Media Assets

deprecated

This endpoint has been deprecated and may be replaced or removed in future versions of the API.

**Please refer to [List Media Assets](/documentation/pr-preview/pr-690/millicast/api/media-assets-list-media-assets.md)**

List media assets, excluding those that have been deleted. A media asset can be:

* A `"FullRecording"` that is from the full duration of the stream
* A `"Clip"` which is generated from a clip request
