# Delete Clip Source Expiry Rule

deprecated

This endpoint has been deprecated and may be replaced or removed in future versions of the API.

**Please refer to [Update Account Media Assets Expiration Rules](/documentation/pr-preview/pr-690/millicast/api/account-update-expiration-rules.md)**

Delete account level expiry rule for clip creation and use system default.

This may immediately trigger removal of existing stream data outside of the system default expiry period, thus preventing clips from being created on those expired streams. Use with caution.
