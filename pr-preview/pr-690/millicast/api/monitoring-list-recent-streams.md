# List Streams

List all streams created within last hour with specific sorting and pagination. If the transcoderId is provided, all streams associated with the current or last active instance will be returned, regardless of the stream creation time. If response array is empty, you have reached the end of the list ordering. Note: This endpoint requires the LiveMonitoring feature.
