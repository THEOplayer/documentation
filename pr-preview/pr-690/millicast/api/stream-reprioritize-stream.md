# Reprioritize an active stream

Update stream priority specified in token and restart stream to re-assign priority.

Use [List Tokens By Name](/documentation/pr-preview/pr-690/millicast/api/publish-token-v-1-list-tokens-by-name.md) and filter by `StreamName` to identify the tokenId associated with the target stream.
