## [📄️Making a custom UI](/documentation/pr-preview/pr-690/open-video-ui/android/guides/custom-ui.md)

[Although the default UI was designed to support a variety of usage scenarios, you may still run into a case that it doesn't handle very well. Perhaps you want to move some buttons around, or add like and dislike buttons to the control bar, or perhaps integrate a text chat component inside your player. In these situations, you may want to build a custom player UI to create a truly unique experience for your viewers.](/documentation/pr-preview/pr-690/open-video-ui/android/guides/custom-ui.md)
