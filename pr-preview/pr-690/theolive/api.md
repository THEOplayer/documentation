# OptiView Live REST API

<!-- -->

OptiView Live provides a REST API to manage channels, receive events, create reports, and set up webhooks.

Alternatively, you can perform the most common actions directly from the OptiView Live dashboard. See [our getting started guide](/documentation/pr-preview/pr-690/theolive/getting-started.md) for more information.

To get started with the API, see the [Authentication](/documentation/pr-preview/pr-690/theolive/api/authentication.md) guide.

The full API specification is also available as a [Swagger document](https://api.theo.live/v2/api-docs/swagger.json).
