# Limitations and known issues

This page lists the current limitations and known issues of OptiView Ads.

## Using OptiView Ads with Picture-in-Picture (PiP) on Firefox for Web[​](#using-optiview-ads-with-picture-in-picture-pip-on-firefox-for-web "Direct link to Using OptiView Ads with Picture-in-Picture (PiP) on Firefox for Web")

Using OptiView Ads with Picture-in-Picture (PiP) on Firefox for Web is currently not supported. During playback, the default PiP button will be hidden to discourage usage.

Note that users will still be able to trigger PiP using the button in the browser's address bar.
