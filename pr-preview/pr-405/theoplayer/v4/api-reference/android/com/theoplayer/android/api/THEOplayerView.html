      • addJavaScriptMessageListener

         void addJavaScriptMessageListener(@NonNull() String type, @NonNull() MessageListener listener)

        Adds a message listener for messages sent from a custom JavaScript file.

        • To send a message from a custom JavaScript file to the the THEOplayerView, call the theoplayerAndroid.sendMessage function from JavaScript.
        • The theoplayerAndroid variable is globally available.
        • Won't have any effect if the player is already destroyed.

        Example:

        theoplayerAndroid.sendMessage('messagetype', 'message');
        Parameters:
        type - The type of messages to listen to.
        listener - The listener that will listen for the messages of the specified type.

      • evaluateJavaScript

         void evaluateJavaScript(@NonNull() String script, @Nullable() ValueCallback<String> resultCallback)

        Asynchronously evaluates JavaScript in the current web context.

        • If non-null, 'resultCallback' will be invoked with any result returned from that execution.
        • This method must be called on the UI thread and the callback will be made on the UI thread.
        Parameters:
        script - the JavaScript to execute, as a String.
        resultCallback - An optional callback to be invoked when the script execution completes with a result of the execution.
