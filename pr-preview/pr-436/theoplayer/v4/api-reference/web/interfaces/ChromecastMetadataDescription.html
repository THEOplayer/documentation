Interface ChromecastMetadataDescription

Describes the metatadata used by Chromecast.

Remarks


- Available since v2.21.0.

    releaseDate?: string;
    releaseYear?: number;
    subtitle?: string;
    title?: string;
    type?: ChromecastMetadataType;
